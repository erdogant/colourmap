Input

The input for colourmap can be either a list of integers or strings.

# Import library
import colourmap as colourmap

# Target labels
y=[1, 1, 2, 2, 3]

# String as input labels
y=["1", "1", "2", "2", "3"]

# Generate colors
c_rgb, c_dict = colourmap.fromlist(y)

Each of these input arrays can be in the form of:

  • List

  • Numpy Array

Output

The output of colourmap is a numpy array with RGB colors that range between [0-1, 0-1, 0-1]. In case of using the fromlist colourmap.colourmap.fromlist(), a dictionary is also returned with the keys as input label and colors as items.
